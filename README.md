# Exercice pratique de la carte aux trésors

## Lancer le projet
* Récupérer le répertoire git
* Lancer votre IDE
    * New project from existing sources
    * Sélectionner le projet que vous venez de récupérer

## Fichier entrant
Le fichier entrant est situé sous `/src/main/resources`.

## Fichier sortant
Le fichier sortant sera généré lors de l'exécution sous le répertoire `/sortie`
